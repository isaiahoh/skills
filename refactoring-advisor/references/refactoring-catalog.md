# Refactoring Technique Index

Each refactoring technique has its own file in `references/techniques/` with full descriptions, step-by-step mechanics, and worked examples from Martin Fowler's *Refactoring* (2nd edition).

**How to use this index**: The smell catalog (`smell-catalog.md`) tells you which techniques to apply for each smell. Look up the technique name below, then read **only the files you need** from the `techniques/` directory. Do not read all technique files — only load the ones relevant to the smells you've identified.
